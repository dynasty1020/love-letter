# love letter

A single-page romantic site, built with React + Tailwind.

## Run locally
```
npm install
npm run dev
```

## Deploy to GitHub Pages
1. Push this folder to a new GitHub repo.
2. Make sure `vite.config.js`'s `base` matches your repo name exactly
   (e.g. `/love-letter/`).
3. Run:
   ```
   npm install
   npm run deploy
   ```
4. In your repo: Settings → Pages → set source branch to `gh-pages`.
5. Your site will be live at `https://<your-username>.github.io/<repo-name>/`.
